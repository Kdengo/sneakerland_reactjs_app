El ratón para juegos SCIMITAR PRO RGB mejora las partidas gracias al sistema de control de botones de macro Key Slider™ personalizable, un sensor óptico con una precisión nativa de 16 000 ppp y almacenamiento de perfiles interno.

SISTEMA EXCLUSIVO KEY SLIDER

Ajuste la posición de los botones hasta 8 mm y bloquéelas con el sistema de control de macros (pendiente de patente) que pone todas las teclas a su alcance, independientemente del estilo de juego. Las teclas laterales con textura mejoran el agarre y el tacto para conseguir una respuesta táctil constante y un control preciso.
